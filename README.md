# AI-Based Sentiment Analyzer

## Description
An AI-powered web application that analyzes the sentiment of user-entered text using Hugging Face Transformers. The application stores sentiment analysis history in MongoDB and allows users to view previous analyses.

## Features
- AI-powered sentiment analysis
- Positive/Negative prediction
- Confidence score
- MongoDB integration
- Analysis history page
- Flask web interface

## Technologies Used
- Python
- Flask
- Transformers (Hugging Face)
- PyTorch
- MongoDB Atlas
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Output

- Analyze text sentiment
- View previous analysis history
- Store results in MongoDB
